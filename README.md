# Calculator
Desktop calculator built with PySide6 featuring dark theme and full keyboard support.

## Requirements
- Python 3.7+
- PySide6

Install dependencies:
```
pip install PySide6
```

## Usage
```
python main.py
```

## Controls
**Mouse:** Click buttons  
**Keyboard shortcuts:**
- Numbers: `0-9`, `.`
- Operations: `+`, `-`, `*`, `/`, `^` (or `p`)
- Execute: `Enter`
- Clear: `Esc`
- Delete: `Backspace`

## Features
- Basic operations (+, -, *, /)
- Exponentiation (^)
- Error handling
- Dark theme
- Calculation history
